# Compartilhamento de Arquivo

```md
# compartilhamento_arquivo

Sistema de compartilhamento de arquivos com upload e gerenciamento via backend.

## ✨ Visão geral

Aplicação full-stack que permite upload, armazenamento e compartilhamento de arquivos de forma simples e eficiente.

## 🚀 Tecnologias

- React
- Vite
- TypeScript
- HTML
- CSS
- Node.js
- Multer
- Express

## 📦 Funcionalidades

- Upload de arquivos
- Download via link
- Interface simples e funcional
```

## ⚙️ Como rodar o projeto

```bash

npm install
npm run dev
```
